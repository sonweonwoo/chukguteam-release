# 축구팀 앱 — 설치 파일 · 홈페이지

이 저장소는 **앱 설치 파일(APK)** 과 **소개 홈페이지(`index.html`)** 를 보관합니다. 앱 소스 코드는 들어있지 않습니다.

### 홈페이지
`index.html` — 앱 소개 랜딩 페이지(정적 HTML, 단일 파일). 브라우저로 바로 열거나 GitHub Pages 등으로 배포할 수 있습니다.

### 설치 주소
**https://chukguteam-api.onrender.com/app**

### 관련 저장소
- 앱 소스 : `sonweonwoo/chukguteam-app` (비공개)
- 서버 소스 : `sonweonwoo/ai-football-team-homepage` (비공개)

### 왜 따로 뒀나
앱 파일(66MB)을 서버 저장소에 넣어오다 보니 저장소가 **3.1GB**까지 불어나
배포가 느려졌습니다. 설치 파일은 여기 두고 서버는 주소만 알려줍니다.
새 빌드가 나오면 이 저장소는 **덮어씁니다**(판본을 쌓지 않아 항상 66MB).
