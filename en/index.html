FAQ

Frequently asked questions

The things people wonder about before installing and getting started.

Is there an iPhone (iOS) version?
Right now we only ship an Android APK. An iOS version is in the works.
Install is blocked with an "unknown sources" warning.
Android requires the "install unknown apps" permission to install an APK from outside the Play Store. From the install prompt, tap through to Settings, allow installs for the browser or file app you're using, then try again.
Is it really free?
Yes — installing and using the app are both free. No purchases or subscriptions.
Do I need to sign up?
A quick sign-up is needed to sync team chat, attendance polls, and schedules. You can start using it right after signing up.
How do updates work?
When a new build ships, the same install URL always serves the latest version. We don't stack old builds — there's always just one up-to-date version.
Is my data safe?
Team data is scoped to your account, and only members of the same team can access it.
